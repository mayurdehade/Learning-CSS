Display Properties

➢ Block Element- Starts on a new line and takes up the full
width available (stretches out to the left and right as far as
it can). Height and width applied. ex. div, p, h1, form.
➢ Inline Element- Does not start on a new line and only takes 
up as much width as necessary. Height and width not applied. 
ex. a, img, span.
➢ Inline-block- Does not start on a new line and only takes up 
as much width as necessary. Height and width not applied.
