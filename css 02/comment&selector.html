Root Selector Function

:root selector function is use for declare global css variables which can we use for defining any style property.
By using global variables we can change major propertes of website with changing only one value not need to change for all places. 


            Syntax:

            :root{
                --primary: any value (ex.red, blue, 15px);
                --secondary: value2;
            }
