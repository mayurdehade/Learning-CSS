Position Properties

    1) static: Default value. Elements render in order, as 
        they appear in the document flow
    2) relative: The element is positioned relative to its normal 
        position, so “left:20px” add 20 pixels to the element’s LEFT 
        position
    3) fixed: The element is positioned relative to the browser
        window
    4) absolute: The element is positioned relative to its first 
        positioned (not static) ancestor element
    5) sticky: The element is positioned based on the user’s scroll 
        position. A sticky element toggles between relative and fixed, 
        depending on the scroll position. It is positioned relative until 
        a given offset position is met in the viewport – then it 
        “sticks” in place (like position: fixed).
