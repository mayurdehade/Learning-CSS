CSS Color Propertes

CSS color can be specified as

1.Predefined color names
Choose color by names different 140 color names available in morder browser.
2.RGB
rgb(red, green, blue) | colour value between 0 and 255 | Black: rgb(0, 0, 0) | White: rgb(255, 255, 255) | Red: rgb(255, 0, 0) | Green: rgb(0, 255, 0) | Blue: rgb(0, 0, 255)
3.RGBA
rgba(red, green, blue, alpha) | Alpha parameter is number between 0.0 (full transparent) to 1 (not transparent at all) | rgba(243, 165, 44, 0.7)
4.HEX code
HEX value start from # | HEX values are between ff to 00 (similar 0-255) | #rrggbb (red, green, blue) | #ff0233
5.HSL
hsl(Hue, Saturation, Lightness) | Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green and 240 is blue | Saturation is a percentage value. 0% means a shade of gray and 100% is the full color | Lightness is also a percentage. 0% is black, 50%is neither light or neither dark, 100% is white | hsl(0, 100%, 50%)
6.HSLA
hsla(Hue, Saturation, Lightness, Alpha) | hsla(120, 80%, 50%, 0.8)
